# JSON2CSV

## Sobre o projeto:
Converter Json para CSV sem bibliotecas ou frameworks, afim de fixar o aprendizado base de JS, HTML5 e CSS3.

## O que aprendemos:

### JAVASCRIPT:
- Json.parse
- Scope (Const, let and var)
- Clean code (Nome de variáveis, funções, etc.)
- Object.keys
- Object.values
- CreateElement
- AppendChild
